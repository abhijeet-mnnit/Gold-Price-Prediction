The project is done majorily in c++ but for the purpose of visualization I have used Wed Devlopment to demonstrate what are the c++ codes trying to do.
The project is deployed via "netlify" with the deploy link "https://shiny-clafoutis-ce5677.netlify.app/". The user has to give their own training set to 
the system in csv file format (for testing purpose a csv file named "gold_train.csv" is given which is the trend of gold price for past 5 years).
After ssuccessful upload of csv file the system calculate the necessary data using Linear Regression and the user got the option to check himself 
for opening price what should be the closing price.
